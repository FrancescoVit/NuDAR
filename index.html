The NUtritional studies Data Analysis Resource

Introduction and rationale of the resource

Nutritional studies (mechanistic, interventional, and epidemiological) make increasingly use of high throughput technologies (e.g. metabolomics, high throughput screening) yielding extremely large datasets. Main strength of these technologies is the ability to describe the system as a whole (systems biology); yet, researchers have encountered major difficulties in mapping these measurements to biological processes.
Part of this problem is caused by the fact that the number of parameters measured greatly out-numbers the number of subjects participating in the study; weakening the power to arise biological conclusions. This power may be enlarged by connecting the outcome of similar studies; however, this requires a great effort by the community of researcher in the nutritional fiels, in agreeing on structured and standardized data storage and methods for integrated analysis.

This website is an effort in this direction. It contains a selection of tools to be used to analyze raw data, produce metadata and annotate metadata from the field of human nutrition, including samples from intervention and observational studies. This website also provides the relevant links to tools used to analyze transcriptomics, proteomics, metabolomics, and the way to annotate and store the metadata produced into a database.

About this resource

This website was developed to provide an open source knowledge base and a complete data querying and analysis platform for nutritional studies.
It is structured in two main pages:

  • The WORKFLOW, where you can find indication and tools for data querying and data analysis procedure. Data analysis procedure will be implemented in a Galaxy instance.
  • The GUIDE, where you can find a list of tool to use in the analysis of different type of data that can be obtained during nutritional studies (ordered by analytical procedure), with a brief description, link to the originl source, and case scenarios applications.

This resource was developed under the working package 4 of the ENPADASI project


About the ENPADASI project

The European Nutritional Phenotype Assessment and Data Sharing Initiative (ENPADASI) is a multidisciplinary consortium of scientists working in the nutritional field.
The main objective of ENPADASI project is to deliver an open access research infrastructure that will contain data from a wide variety of nutritional studies, ranging from mechanistic/interventions to epidemiological studies including a multitude of phenotypic outcomes that will facilitate combined analyses in the future.
To obtain further information on the tasks, objective and organization of the project, to know the partners involved in ENPADASI, or to be updated on the results, have a look at the project website.
